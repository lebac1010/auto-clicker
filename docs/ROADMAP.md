# Roadmap

Version: 1.0
Date: 2026-02-09

## 0. Assumptions
1. Team: 2 Flutter devs, 1 Android native dev, 1 QA part-time, 1 BA or PM part-time.
2. Work week: 5 days.
3. Parallel work is possible after Milestone 1.

## 1. Milestone 0 - Project Setup
Duration: 1 week
Staffing: Flutter x1, Android x1
1. Flutter project scaffold
2. Android native module setup
3. Build and run on device

## 2. Milestone 1 - Core Permissions and Home
Duration: 1.5 weeks
Staffing: Flutter x2, Android x1, QA x0.5
1. Splash and routing
2. Onboarding
3. Permissions Hub
4. Home with quick actions

## 3. Milestone 2 - Script Engine and Editor
Duration: 2 weeks
Staffing: Flutter x2, Android x1, QA x0.5
1. Script model and storage
2. Editor for multi-point taps
3. Validation and save

## 4. Milestone 3 - Floating Controller
Duration: 2 weeks
Staffing: Flutter x1, Android x1, QA x0.5
1. Overlay bubble and panel
2. Marker rendering and drag
3. Start, pause, stop integration

## 5. Milestone 4 - Recorder
Duration: 2 weeks
Staffing: Flutter x1, Android x1, QA x0.5
1. Record actions and delays
2. Timeline editor
3. Save as script

## 6. Milestone 5 - Import and Export
Duration: 1 week
Staffing: Flutter x1, Android x0.5, QA x0.5
1. JSON export
2. JSON import
3. Validation and error states

## 7. Milestone 6 - Stability and QA
Duration: 2 weeks
Staffing: Flutter x1, Android x1, QA x1
1. OEM-specific guides
2. Error recovery
3. Performance tests

## 8. Total Estimate
1. Calendar time: 11.5 weeks.
2. Buffer recommended: 10% for OEM issues and policy changes.
